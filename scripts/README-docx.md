# docx → HTML 변환 (갑목 등 천간별 성향 페이지용)

Word 문서(docx)의 **굵기·글자 크기**를 유지한 채 HTML로 바꿔서 갑목 페이지에 넣을 수 있습니다.

## 방법

1. **터미널**을 연 다음, 프로젝트 폴더(웹사이트)로 이동합니다.

2. 아래처럼 docx **전체 경로**를 넣고 실행합니다.

   ```bash
   node scripts/docx-to-html.cjs "C:\Users\titko\OneDrive\바탕 화면\오리엔탈 포캐스트\홈페이지\천간별 성향 설명\1.갑목 성향 3.docx"
   ```

   (경로에 한글이 있으면 위처럼 큰따옴표로 감싸 주세요.)

3. 터미널에 출력된 **HTML 전체**를 복사합니다.

4. `src/app/personality/gapmok/page.tsx`를 열고, **`GAPMOK_HTML`** 안의 기존 HTML을 지운 뒤, 3번에서 복사한 내용으로 **그대로 붙여넣기** 합니다.  
   (백틱 문자열 안에 넣으면 됩니다.)

5. 저장하면 갑목 페이지에 docx와 같은 굵기·글자 크기가 반영됩니다.

## docx를 프로젝트 안에 둔 경우

예: `1.갑목 성향 3.docx`를 프로젝트 루트에 복사해 뒀다면:

```bash
node scripts/docx-to-html.cjs "1.갑목 성향 3.docx"
```

이렇게 상대 경로로 실행해도 됩니다.
